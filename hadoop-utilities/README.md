# hadoop utilities

This is a set of utilities that aims to simpifly Java interaction with ArcGIS formats, such as the JSON representaiton of a feature class.  

## Featues
* API methods/classes for reading JSON into a set of Java objects that describe a feature class. 

## Requirements

* Java JDK 1.6 or greater
